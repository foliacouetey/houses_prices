# houses_prices

This project aims to predict houses prices using machine learning techniques. It includes implementations in both Python and R.

## Project structure

```

houses_prices/
├──── .gitignore
├──── LICENSE
├──── README.md
├──── PYTHON/
│      ├──── requirements.txt
│      ├──── data/
│      │       ├──── train/
│      │       │       └──── train.csv
│      │       └──── test/
│      │               └──── test.csv
│      └──── scripts/
│              └──── main.ipynb
└──── R/
      ├──── renv.lock
      ├──── R.Rproj
      ├──── data/
      │       ├──── train/
      │       │       └──── train.csv
      │       └──── test/
      │               └──── test.csv
      └──── scripts/
              └──── main.R

```
